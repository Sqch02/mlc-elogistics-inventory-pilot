## Produits et Stock
- Table SKUs (stock actuel, conso 30j, moyenne 90j, jours restants)
- Détail SKU: courbe conso, réassorts

## Expéditions
- Liste filtrable par date, transporteur
- Détail expédition: poids, coût estimé, items

## Emplacements
- Liste emplacements + SKU assigné
- Recherche SKU -> emplacement

## Pricing

- Import et validation (tarifs manquants)

## Facturation
- Sélection mois
- Génération récap
- Export CSV

## Réclamations
- Liste + filtres
